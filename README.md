# Word-Search

7x7 보드 워드 서치 웹 게임입니다.

## 사용 방법

1. `index.html`을 브라우저에서 엽니다.
2. 보드에서 글자를 드래그하여 단어를 찾습니다.
3. `words.txt`에 단어 목록을 한 줄씩 추가하면 게임이 자동으로 불러옵니다.
4. 게임 시작 시 4~5자 단어 몇 개를 보드에 미리 배치하고 나머지를 랜덤으로 채웁니다.
5. 보드 생성이 느릴 경우 Live Server로 실행하면 안정성이 높아집니다.

## 파일

- `index.html` : 웹 게임 화면
- `style.css` : 보드 스타일 및 레이아웃
- `script.js` : 단어 검색 로직, 보드 생성, 드래그 입력 처리
